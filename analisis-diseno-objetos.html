Análisis y Diseño Orientado a Objetos

Diagrama UML

Información del Diagrama UML

1. EmpresaLuz

Atributos:

  • nombreE: string
  • nroEmpleados: int
  • empleados: vector<Empleado>
  • clientes: vector<Cliente>

Funciones:

  • actualizarServicio()
  • cortarServicio()

Conexiones: Relación de composición con Empleado y Cliente.

2. Factura

Atributos:

  • idFactura: string
  • idCliente: string
  • fechaEmision: string
  • fechaVencimiento: string
  • montoTotal: float

Conexiones: Asociación con Cliente.

3. Cliente

Atributos:

  • idCliente: string
  • sugerencias: string
  • facturas: vector<Factura>

Funciones:

  • realizarPago()

Conexiones: Herencia de Persona. Asociación con Factura y CuentaCliente.

4. Persona (Clase base)

Atributos:

  • nombre: string
  • id: int
  • direccion: string
  • telefono: int
  • email: string

5. RecursoHumano

Atributos:

  • consultas: vector<string>

Funciones:

  • hacerConsulta()

6. Empleado

Atributos:

  • sueldo: int
  • idEmpleado: string
  • herramientas: vector<string>

Funciones:

  • hacerMantenimiento()
  • cobrar()

Conexiones: Herencia de Persona. Asociación con Mantenimiento.

7. CuentaCliente

Atributos:

  • idCliente: string
  • idCuenta: string
  • estadoCuenta: string
  • tipoServicio: string
  • deudas: bool

8. ServicioLuz

Atributos:

  • idMedidor: string
  • saldo: int
  • estadoDeudas: bool

Funciones:

  • actualizarServicio()
  • cortarServicio()

9. Medidor

Atributos:

  • idMedidor: string
  • idCliente: string
  • fechaLectura: string
  • valorLectura: int
  • ubicacion: string
  • fechaInstalacion: string
  • estado: string

Funciones:

  • realizarLectura()
  • calcularSaldo()
  • getEstado()

10. Mantenimiento

Atributos:

  • sueldo: int
  • idEmpleado: string
  • herramientas: vector<string>

Funciones:

  • hacerMantenimiento()

Conexiones: Asociación con Empleado.

Conexiones Principales

  • EmpresaLuz tiene relaciones de composición con Empleado y Cliente.
  • Cliente está asociado a Factura y CuentaCliente.
  • Empleado está asociado a Mantenimiento.
  • ServicioLuz se conecta con Medidor.
